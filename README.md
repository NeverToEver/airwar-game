```bash

```

## 游戏操作

### 基础控制

| 按键 | 功能 |
|------|------|
| `W/A/S/D` 或 `↑←↓→` | 移动战机 |
| `空格` | 发射子弹（长按连续射击）|
| `ESC` | 暂停游戏 |

### 特殊功能

| 按键 | 功能 | 说明 |
|------|------|------|
| `H` (长按3秒) | 🚁 进入母舰 | 清除敌机 + 保存进度 |
| `K` (长按3秒) | 💀 放弃游戏 | 触发死亡动画 + 结束菜单 |

### 🎁 增益系统 (Roguelike)

每达成里程碑分数时，从奖励池中选择增益效果：

#### 攻击类增益

| 增益 | 图标 | 效果 |
|------|------|------|
| Power Shot | DMG | 子弹伤害 +25% |
| Rapid Fire | RPD | 射击间隔 -20% |
| Piercing | PIR | 子弹穿透 1 个敌人 |
| Spread Shot | SPD | 同时发射 3 颗子弹 |
| Explosive | EXP | 范围伤害 30 点 |

#### 防御类增益

| 增益 | 图标 | 效果 |
|------|------|------|
| Extra Life | HP | 最大生命 +50 |
| Regeneration | REG | 每秒回复 2 点生命 |
| Lifesteal | LST | 击杀回复生命 |
| Shield | SHD | 抵挡下一次攻击 |
| Armor | ARM | 受到伤害 -15% |
| Evasion | EVD | 20% 闪避几率 |

### 🎯 Boss 机制

- **生成间隔**: 每 1800 帧 (~30秒)
- **逃跑惩罚**: 逃跑后间隔延长至 2700 帧 (~45秒)
- **攻击模式**: 扇形弹幕 / 追踪弹 / 全方位弹幕
- **进化阶段**: 每 300 帧升级，攻击更凶猛

### 设计原则

| 原则 | 描述 | 实施状态 |
|------|------|----------|
| **单一职责 (SRP)** | 每个类只负责一件事 | ✅ 完全遵循 |
| **开闭原则 (OCP)** | 对扩展开放，对修改关闭 | ✅ 策略模式 |
| **里氏替换 (LSP)** | 子类可替换父类 | ✅ 接口一致 |
| **依赖倒置 (DIP)** | 依赖抽象而非具体 | ✅ Protocol 定义 |
| **接口隔离 (ISP)** | 客户依赖最小接口 | ✅ IBulletSpawner |
| **组合优于继承** | 使用组合代替继承 | ✅ Manager 模式 |
