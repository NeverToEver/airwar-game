# 飞机大战 - 待测试清单

本文档列出游戏中所有需要手动测试的功能点。建议在每次更新后逐一验证。

## 一、基础功能测试

### 1.1 登录系统
- [ ] 可以注册新账号
- [ ] 可以登录已有账号
- [ ] 登录后用户名正确显示
- [ ] 登录失败显示错误提示
- [ ] 用户数据正确保存到数据库

### 1.2 菜单系统
- [ ] 难度选择可以切换（Easy/Medium/Hard）
- [ ] ESC 可以从菜单返回登录界面
- [ ] Enter 可以开始游戏
- [ ] 难度选项显示正确的子弹数

### 1.3 游戏主循环
- [ ] 玩家可以正常移动（WASD/方向键）
- [ ] 空格键可以发射子弹
- [ ] 玩家离开屏幕边界后被阻挡
- [ ] 玩家有闪烁的无敌帧效果（受击后）

### 1.4 敌人系统
- [ ] 敌人从屏幕顶部生成
- [ ] 敌人向下移动
- [ ] 敌人离开屏幕后消失
- [ ] 不同难度生成速度不同

### 1.5 碰撞检测
- [ ] 玩家子弹击中敌人，敌人受伤
- [ ] 敌人子弹击中玩家，玩家受伤
- [ ] 玩家与敌人碰撞，玩家受伤
- [ ] **玩家受击体积比实际精灵小**（20x24 vs 50x60）
