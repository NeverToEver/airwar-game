# Air War 军事风格 HUD UI 设计

## 概述

为 Air War 飞机大战游戏设计现代化的军事风格 UI，提升游戏视觉体验。设计语言基于硬核军事显示器风格，强调精密仪器感和战场氛围。

## 设计语言

### 色彩系统

| 用途 | 颜色名 | Hex 值 | 说明 |
|------|--------|--------|------|
| 主色 | AMBER_PRIMARY | #FFB432 | 琥珀色，主要信息色 |
| 主色 | AMBER_DIM | #CC9026 | 琥珀色暗淡，用于次要信息 |
| 强调 | MILITARY_GREEN | #4A7C59 | 军事绿，状态指示 |
| 背景 | BG_PRIMARY | #0A0A0F | 近黑色，主背景 |
| 背景 | BG_PANEL | #12141A | 面板背景 |
| 边框 | BORDER_GLOW | #FFB43266 | 发光边框（半透明） |
| 危险 | DANGER_RED | #FF4444 | 危险/低血量 |
| 文字 | TEXT_PRIMARY | #FFE4B5 | 主要文字 |
| 文字 | TEXT_DIM | #8B8B7A | 次要文字 |

### 几何元素

**切角边框 (Chamfered Border)**
- 45° 斜角切角
- 切角深度: 12px
- 边框线宽: 2px
- 发光效果: 1px 半透明外晕

```
  ┌──────────────┐
  │╲            ╱│
  │  ╲        ╱  │
  │    ╲    ╱    │
  │    ╱    ╲    │
  │  ╱        ╲  │
  │╱            ╲│
  └──────────────┘
```

### 动效规范

| 动效 | 参数 |
|------|------|
| 扫描线 | 垂直移动，速度 2s/周期，alpha 0.05 |
| 边框脉冲 | 频率 1Hz，alpha 0.3-0.6 |
| 悬停放大 | scale 1.0 → 1.02，时间 150ms |
| 信息更新闪烁 | alpha 1.0 → 0.5 → 1.0，时间 200ms |

## 模块设计

### 1. HUD (抬头显示)

**布局**: 顶部居中 + 底部两侧

**顶部 HUD 条**
```
┌─────────────────────────────────────────────────────────────┐
│╲  SCORE: 12,500    MILESTONE: 80%    KILLS: 47    ╱│
└─────────────────────────────────────────────────────────────┘
```
- 切角边框包裹
- 左侧: 分数 + 进度
- 右侧: 连杀数 + BOSS击杀
- 中文字使用紧凑字间距

**底部左侧: 生命值**
```
┌────────────────────┐
│╲ ❤️ ████████░░ 80% │
└────────────────────┘
```
- 心形图标 + 血条 + 百分比
- 血条分段: 每 10% 一小格
- 低血量时(<30%) 血条变红并脉冲

**底部右侧: 难度系数**
```
┌────────────────────┐
│╲ COEFF: 1.5x      │
│   DIFFICULTY: MED  │
└────────────────────┘
```

### 2. 菜单系统

**主菜单面板**
```
┌────────────────────────────────┐
│╲                              ╱│
│  ╔════════════════════════╗   │
│  ║      A I R  W A R     ║   │  ← 六边形标题框
│  ╚════════════════════════╝   │
│                                │
│      ▶ NEW GAME               │  ← 箭头指示器
│        CONTINUE               │
│        SETTINGS               │
│        TUTORIAL                │
│        QUIT                    │
│╱                              ╲│
└────────────────────────────────┘
     ↗ 微弱网格背景 + 扫描线
```

- 面板使用切角边框
- 背景: 近黑色 + 微弱网格线 (alpha 0.1)
- 扫描线效果覆盖整个菜单
- 选项悬停: 边框亮起 + 箭头放大
- 选项点击: 快速闪烁反馈

**暂停菜单**
```
┌────────────────────────────────┐
│╲        P A U S E D           ╱│
├────────────────────────────────┤
│╲      ▶ RESUME                ╱│
│        SAVE AND QUIT          │
│        QUIT WITHOUT SAVING     │
│╲                              ╱│
└────────────────────────────────┘
```

### 3. Boss 血条

**位置**: 屏幕顶部

```
┌─────────────────────────────────────────────────────────────┐
│╲ BOSS: NEMESIS CRUISER                    PHASE 2/3       ╱│
│╲ [██████████████░░░░░░░░░░░░░] 67%                        ╱│
└─────────────────────────────────────────────────────────────┘
```

- 宽幅切角边框
- 血条分段显示 (每 25% 一段)
- Phase 指示器在右侧
- 血量变化时有平滑动画
- Phase 切换时边框脉冲发光

### 4. Buff 状态面板

**位置**: 屏幕右侧

```
┌─────────────┐
│╲ ACTIVE     ╱│
├─────────────┤
│╲ ⚡ PWR +2  ╱│
│╲ 🛡 DEF +1  ╱│
│╲ 💨 SPD +3  ╱│
└─────────────┘
```

- 六边形小图标 (24x24)
- Buff 名称 + 等级
- 激活时边框发光
- 满级时金色高亮

### 5. Milestone 进度条

**位置**: 屏幕底部中央

```
┌─────────────────────────────────────────────┐
│╲ NEXT REWARD IN: ████████░░░░░░░ 67%       ╱│
└─────────────────────────────────────────────┘
```

- 显示距下一个里程碑的进度
- 到达 100% 时边框闪烁并弹出奖励选择
